FLSVisitourRestAPI API documentation version 0.0.3

http://km.aifb.kit.edu/services/bader4/step

This Web API is an RESTful Wrapper for the FLS visitour server. It allows to create, read, update and delete Work Orders for field technicians as examined in the STEP (https://www.projekt-step.de/en/) research project. All interactions are state-based and purly RESTful. Creating a new Work Order will make the server plan a new job for the best suited technician and propose three possible appointments. Overwriting an Work Order and setting an appropriate state (recorded, confirmed etc.) results in an update of the tour planning. Be aware: Side-effects DO happen, as long as Work Orders are not confirmed! More information can be found at https://www.fastleansmart.com/anwendungen/tourenplanung/
